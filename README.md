## ComfyUI Startup

### 简介

ComfyUI Startup是一个ComfyUI启动器，目前社区上大部分都是只支持Windows的启动器。为此特地开发了一个支持Macos的启动器，并优化了国内网络问题，可以很方便的下载插件和模型。

### 技术栈

本项目使用Tauri跨平台框架开发。

**前端使用**

- React
- Zustand
- React-router

**后端使用**

- Tauri
- Rust
- Tokio
